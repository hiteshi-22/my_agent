# LLM Agents Project 
This repository contains AI agents built with Google's Generative AI and Tavily search. The agents can answer user queries by combining LLM capabilities with external tools. --- 
## Agents Covered in this project
1. Basic Agent
2. Smart Agent
3. Calculator Agent
4. Movie Agent

Environment Variables
The environment variables placeholders are mentioned in the .env.example. You can  create your own env file by giving the API keys. API key can be created by following the below steps.

API Initialization
## Google API Key
Go to Google AI Studio or your Google Cloud Console.
Create a new API key.
Copy the key and keep it safe.

## Tavily API
Sign up at Tavily.
Generate a developer API key from your account dashboard.
Copy the key.

## OMDB API Key
Sign up at OMDB API
